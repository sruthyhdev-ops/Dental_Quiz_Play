# 🦷 Dental Exam Quiz

A professional, minimalist Android application designed for OPSC Dental Surgeon aspirants and BDS students. Built with a focus on speed, accuracy, and high-yield revision.


## 🚀 Features

- **Vast Question Bank:** Over 600+ questions curated from OPSC 2014, 2018, and 2021 official papers.
- **Marathon Mode:** Simulate real exam endurance with 50, 100, 150, or 200 question sessions.
- **Quick Quiz:** Rapid-fire 20-question sets for daily revision.
- **High-Yield Flashcards:** 150+ swipable cards covering core dental subjects.
- **Exam Simulation:**
  - **Difficulty Levels:** Easy (60s), Medium (45s), and Hard (30s) per question.
  - **Negative Marking:** Optional -0.25 penalty to mirror OPSC standards.
  - **Instant Feedback:** Toggle between learning mode and mock-exam mode.
- **Reference Library:** Quick access to recommended BDS textbook lists.

## 🛠️ Tech Stack

- **Frontend:** React / Vite / Tailwind CSS
- **Backend/Auth:** [Supabase](https://supabase.com/)
- **Monetization:** Google AdMob (Rewarded & Interstitial)
- **Deployment:** Capacitor (Android)
